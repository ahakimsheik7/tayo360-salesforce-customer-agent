# Tayo360 Salesforce Customer Engagement Agent

## Overview

Tayo360 Customer Engagement Agent is a Salesforce-powered AI customer-facing agent designed to help schools, childcare centers, organizations, and community partners learn about Tayo360 services.

The agent supports two major business areas:

1. Tayo360 Social Development Series V1: Understanding Boundaries
2. Tayo360 AI Consulting Services

The project is designed as a portfolio demonstration, business automation system, and grant-ready proof-of-concept showing how AI can improve access to educational resources and business consultation.

## Core Offers

### Social Development Series V1
- Understanding Boundaries book
- Curriculum package
- Teacher guide
- Parent engagement guide

### AI Consulting Services
- AI receptionist
- AI workflow automation
- AI agents
- Website and business automation consultation

## Target Users

- Schools
- Childcare centers
- District leaders
- Nonprofit organizations
- Community programs
- Small businesses interested in AI automation

## Main Channels

- Salesforce-hosted web agent
- Tayo360 website chat
- WhatsApp
- Phone/AI receptionist

## Project Goals

- Capture qualified leads
- Answer common questions
- Recommend the right Tayo360 service
- Book consultations
- Store inquiries in Salesforce
- Demonstrate measurable community and business impact
- Support future grant applications

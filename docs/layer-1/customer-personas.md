# Customer Personas

## Persona 1: School Director

Goals:
- Improve student behavior
- Provide structured curriculum
- Support teachers

Needs:
- Easy-to-use program
- Proven results
- Parent involvement tools

Questions:
- How does this fit into classrooms?
- How much does it cost?
- Can teachers use this easily?

- Simple implementation
